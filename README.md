# in-geveb-corpus

Corpus of Yiddish based on https://ingeveb.org

Last updated: 8/12/2018

## Contents
- Various texts in Yiddish, broken down to sentences
- Each sentence available in multiple orthographies
    - YIVO
    - Chassidic
    - Soviet
    - Romanized/Latin alphabet

## Directory structure

.
